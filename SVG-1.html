SVG

Use SCG to draw vector graphic images into a page. You can create shapes like circles and rectangles. You can also create polygons shapes and lines. Objects can have a stroke and a fill.

SVG objects are created with "tags" like other HTML elements. Most fo the SVG tags use attributes to set their appearance, elements can also be styled and animated with CSS.

SVG is written in XML, similar to HTML it follows slightly different rules. In the SVG element all elements must close, and if they don't have closing tag the element should end if the /. For example <circle />

  • cx - set the x or horizontal position of the element
  • cy - set the y or vertical position of the element
  • r - set the radius of the circle
  • stroke - set the color of the stroke
  • stroke-width - set the weight of the stroke
  • fill - set the fill color

An SVG "drawing" is contained in the SVG tag. Use the width and height attributes to set the size of the drawing.

The circle is located around the center point. Set the cx, and cy to 0 and refresh. You will see the circle aligned with the upper left corner of the SVG element. Notice the cx, and cy values are relative to the SVG element.
